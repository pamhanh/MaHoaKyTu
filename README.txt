MaHoaKyTu
=========

Demo ma hoa Caesar va Vigenere.

Chay demo GitHub:
https://pamhanh.github.io/MaHoaKyTu/

Chay tren may tinh:

1. Cai thu vien:
   pip install -r requirements.txt

2. Chay app:
   python app.py

3. Mo trinh duyet:
   http://127.0.0.1:5000/

Dung app:
Nhan Ctrl + C trong cua so terminal dang chay app.py.
